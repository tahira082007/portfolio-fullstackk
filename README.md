# Portfolio Full Stackk Website

This is a full stack portfolio website built using:

Frontend: HTML, CSS, JavaScript  
Backend: Node.js (Express)  
Database: MySQL  

Features:
- Contact form connected to backend
- Data stored in MySQL database
- Responsive and modern UI

How to run:
1. Start XAMPP (MySQL)
2. Run backend using node server.js
3. Open frontend index.html
